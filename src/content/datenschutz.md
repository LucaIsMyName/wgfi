## 3. Zugriffsdaten und Hosting

Der Wiener Grünflächen Index ist eine statische Webanwendung. Es werden **keine personenbezogenen Daten auf unseren Servern gespeichert**. Ihr Webbrowser stellt beim Aufruf der Website eine Verbindung zum Hosting-Anbieter her. Dabei können technische Zugriffsdaten (IP-Adresse, Datum und Uhrzeit, aufgerufene Seite) kurzzeitig in Server-Logs gespeichert werden. Diese Daten dienen ausschließlich der Sicherstellung des Betriebs und werden nicht an Dritte weitergegeben.

## 4. Lokaler Speicher (localStorage)

Diese Anwendung verwendet den lokalen Speicher Ihres Browsers (localStorage / sessionStorage), um folgende Daten **ausschließlich auf Ihrem Gerät** zu speichern:

- **Favoriten**: Parks, die Sie als Favoriten markiert haben.
- **Suchfilter**: Zuletzt verwendete Filter und Suchbegriffe.
- **Designeinstellungen**: Gewähltes Farbthema (Hell/Dunkel).
- **Besuchshistorie**: Zuletzt angesehene Parks.

Diese Daten verlassen Ihr Gerät zu keinem Zeitpunkt und werden nicht an Dritte übertragen. Sie können diese Daten jederzeit über die Einstellungen Ihres Browsers löschen.

Beim Laden der Karte wird eine Verbindung zu den Servern von Mapbox hergestellt. Dabei wird Ihre IP-Adresse übertragen. Mapbox verarbeitet diese Daten gemäß seiner eigenen Datenschutzerklärung, die unter [mapbox.com/legal/privacy](https://www.mapbox.com/legal/privacy) abrufbar ist.

Die Kartenansicht wird nur geladen, wenn Sie die Kartenseite aufrufen.
